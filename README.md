Usage: 
  Tga2bti <format> <infile> <outfile>

Formats:

  -I4   =  4-bit Intensity

  -I8   =  8-bit Intensity

  -IA4  =  4-bit Intensity + 4-bit Alpha

  -IA8  =  8-bit Intensity + 8-bit Alpha

  -565  = 16-bit RGB

  -4A3  = 16-bit RGBA

Note:
  Only 24 and 32 bit uncompressed TGA files are supported
  All Intensity formats will be saved in grayscale
